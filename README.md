### Linkedlist and trie implementation in C++

### Building the project

- clone the project
    git@github.com:SarthakMakhija/C-collections.git

- create a build directory
    mkdir build

- change to build directory
    cd build

- execute CMake
    cmake --build . -v

### Running tests

- change to build directory
    cd build

- execute collections_tests
    ./collections_tests
